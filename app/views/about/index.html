About

TrackMyCosts.com is a service built by OpenNode team opennodecloud.com for making tracking of expenses easier.

At OpenNode we are using quite a lot of services - AWS, Azure, Digital Ocean, Google Apps, On-Demand Jira and so on. We use them to run multiple projects. The issue we had was understanding if expenses actually made sense.

TrackMyCosts.com is built on top of the open-source platform NodeConductor https://github.com/opennode/nodeconductor/ and allows to analyse expenses by allocating resources from different cloud providers to the projects you create.
