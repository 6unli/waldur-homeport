Our features

Actionable events

All events of the system are recorded and timestamped. You have the full audit trail. NodeConductor analyses logs and provides a convenient way to act in case a human intervention is required.

Full monitoring

System and application monitoring. Heartbeat of project activity.

Reliable backups

Make sure that your infrastructure is safe by setting up regular backups. Don't bother with building your backup solution from scratch.

Cloud-agnostic

OpenStack, VMware, AWS, Rackspace or a standalone server — the choice is yours.

One-click deployment

Use pre-configured templates to quickly setup your multi-component development or production environment.

Managing resource access

Working on a project in a team? Simply assign roles to collaborators and they will get access to all the needed resources at once.

Plans

Plans are defining limits for the setup managed by the NodeConductor.

  • Projects contain resources that belong to a organization and are shared among a defined set of users.
  • Resource is an entity managed by NodeConductor — for example, it may be a virtual machine on OpenStack, a Git repository on Github or a project in Atlassian JIRA.
  • Users are people that are working in your projects.
