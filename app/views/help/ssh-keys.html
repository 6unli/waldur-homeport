How to generate RSA keys

Ubuntu

On your machine create public and private SSH keys using the command-line:
mkdir ~/.ssh
chmod 700 ~/.ssh
ssh-keygen -t rsa

You will be asked to confirm keys location and create passphrase to protect your private key: 

 Generating public/private rsa key pair.
 Enter file in which to save the key (/home/b/.ssh/id_rsa):
 Enter passphrase (empty for no passphrase):
 Enter same passphrase again:
 Your identification has been saved in /home/b/.ssh/id_rsa.
 Your public key has been saved in /home/b/.ssh/id_rsa.pub.

After successful creation you should copy id_rsa.pub (your public key) content and paste it into "Public key" input on Add key page
For other Linux systems search instructions for your distribution.

OS X

Instructions are pretty much the same as for Linux,
but you may omit directory creation and enter only this line 

 ssh-keygen -t rsa

Windows

On of the possible ways to create SSH keys on Windows is using Git Bash. This set of steps is based this tool, you need to install it to proceed further.

At first check if you already have public key, if so you can use it on our Add Key page. Open Git Bash and enter: 

 ls -al ~/.ssh
This command will show existing files in your .ssh directory. Public key has .pub extension (usually id_rsa.pub)

To generate new SSH key pair in your Git Bash enter: 

 ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
This will create new ssh key, using the provided email as a label, when you see information about choosing keys directory just press Enter.
You will also be asked to enter a passphrase. After all is done you will see output like this: 
 # Your identification has been saved in /Users/you/.ssh/id_rsa.
 # Your public key has been saved in /Users/you/.ssh/id_rsa.pub.
 # The key fingerprint is:
 # 01:0f:f4:3b:ca:85:d6:17:a1:7d:f0:68:9d:f0:a2:db your_email@example.com

Next, you have to enable your ssh-agent and add keys that you have created.
Open your Git Bash and start the agent: 

 # start the ssh-agent in the background
 ssh-agent - s
Add your SSH key to ssh-agent: 
 ssh-add ~/.ssh/id_rsa
That's it, you are ready to go!
